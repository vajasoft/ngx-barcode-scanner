# Vajasof Barcode Scanner

## Build for GitHub Pages
```sh
ng build ngx-barcode-scanner
ng build demo --output-path docs --base-href /ngx-barcode-scanner/
```

## Serve
```sh
ng serve --host 0.0.0.0
```

## NPM
```sh
npm login
npm publish
```

if scoped (@vajasoft)
```sh
npm publish --access public
```
